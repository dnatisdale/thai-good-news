🔍 What We'd Like You to Evaluate

Please spend some time exploring the app and consider the following areas:

  1. Features & Functionality — Do the core features work as expected? Try browsing languages, playing audio, searching, saving favorites, and downloading for offline use.
  2. User Experience — Is the app intuitive and easy to navigate? Does everything feel polished and responsive?
  3. Content & Presentation — Is the information presented clearly in both English and Thai? Are there any typos or translation issues?
  4. Missing Features — Is there something you expected to find but didn't? What "should" be in the app but isn't?
  5. Unnecessary Features — Does anything feel redundant or like overkill? Would removing it improve the experience?
  6. Performance — Does the app load quickly? Does audio playback work smoothly? How does it handle offline mode?

✨ Key Features to Explore

  • Language Finder — Browse and search 100+ languages in Thai or English
  • Audio Player — Listen to Gospel recordings with full playback controls
  • Offline Support — Download messages for use without internet
  • Search — Find content by language, title, or scripture reference
  • Favorites & Notes — Save messages and write personal notes
  • QR Code Cards — Generate and print shareable QR cards for outreach
  • Sign Language Video — Thai Bible Sign Language with offline viewing
  • Share — Share via email, LINE, clipboard, or native device sharing
  • Import Content — Add new recordings from the GRN database
  • Dark Mode — Toggle between light and dark themes
  • Bilingual Interface — Switch between English and Thai instantly
